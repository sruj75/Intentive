# Incremental Scope Through MVPs

Intentive will be scoped and built through small MVP loops rather than fully planned up front. The current sequence is Chat Shell, Continuity, Follow-Up Objects, then Boundaries, with V1 focused on relational chat, V1.5 focused on the proactive loop, and V2 focused on broader autonomous agency. This matches the product reality that the right interaction model will be discovered by building and observing, not by designing every behavior in advance.

**Consequences**

- PRDs and issues should describe independently buildable slices.
- Each slice should preserve the product spine: companion chat, continuity, gentle proactivity, and capability honesty.
- Later planning should be allowed to revise details, but should not silently overturn the V1/V1.5/V2 scope split without an explicit decision.
