# Chat-First Mobile Surface

V1 opens into one continuous companion chat instead of a dashboard, task board, or multi-tab productivity workspace. The deliberate trade-off is that early structure stays mostly hidden so Intentive can feel like a living companion surface: fast to open, emotionally low-friction, and centered on continuity rather than management.

**Considered Options**

- Dashboard-first productivity app.
- Multi-tab app with chat, tasks, memory, and settings as peers.
- Single-thread companion chat with secondary structure revealed only when useful.

**Consequences**

- V1 design should prioritize conversation ergonomics, presence, and continuity cues.
- Tasks, reminders, memory, and settings should initially appear as lightweight conversational affordances or sheets, not primary navigation.
- The app should avoid streaks, dense task furniture, and generic productivity-app patterns unless later evidence shows users need them.
